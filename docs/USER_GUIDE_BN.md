## পরিচয়

**অ্যাক্সেসিবিলিটি অটোমেশন এজেন্ট** হল একটি কৃত্রিম বুদ্ধিমত্তা চালিত ডেস্কটপ অটোমেশন টুল যা হাত ছাড়াই বা সীমিত গতিশীলতা সহ ব্যবহারকারীদের জন্য ডিজাইন করা হয়েছে। এটি নিম্নলিখিত মাধ্যমে আপনার কম্পিউটারের সম্পূর্ণ নিয়ন্ত্রণ সক্ষম করে:

- **ভয়েস কমান্ড**: আপনার কম্পিউটার নিয়ন্ত্রণ করতে প্রাকৃতিক কমান্ড বলুন
- **চোখের ট্র্যাকিং**: আপনার চোখ দিয়ে মাউস কার্সার নিয়ন্ত্রণ করুন
- **চোখের পলক সনাক্তকরণ**: চোখ ঝাপসা করে ক্লিক করুন
- **ম্যাক্রো রেকর্ডিং**: জটিল কর্মপ্রবাহ রেকর্ড এবং পুনরাবৃত্তি করুন
- **ব্রাউজার অটোমেশন**: ওয়েব পেজ অনুসন্ধান, নেভিগেট এবং ইন্টারঅ্যাক্ট করুন
- **অ্যাপ্লিকেশন লঞ্চার**: ভয়েস কমান্ড দিয়ে অ্যাপ্লিকেশন লঞ্চ করুন
- **GUI সনাক্তকরণ**: স্বয়ংক্রিয়ভাবে স্ক্রিনে ইন্টারঅ্যাক্টিভ উপাদান সনাক্ত করুন

### মূল সুবিধা

✅ **হাত ছাড়াই নিয়ন্ত্রণ**: কীবোর্ড বা মাউস ছাড়াই সম্পূর্ণ ডেস্কটপ অটোমেশন
✅ **দ্বিভাষিক সমর্থন**: ইংরেজি এবং বাংলা ইন্টারফেস এবং ভয়েস কমান্ড
✅ **কম স্পেক বান্ধব**: 4GB RAM এবং Intel i3 প্রসেসর সহ কম্পিউটারের জন্য অপ্টিমাইজ করা
✅ **ত্রুটি পুনরুদ্ধার**: স্বয়ংক্রিয়ভাবে ব্যর্থতা থেকে পুনরুদ্ধার করে
✅ **কাস্টমাইজযোগ্য**: আপনার কর্মপ্রবাহের জন্য কাস্টম ম্যাক্রো রেকর্ড এবং সংরক্ষণ করুন

## ইনস্টলেশন

### ধাপ 1: Python ডাউনলোড এবং ইনস্টল করুন

1. [python.org](https://www.python.org/downloads/) পরিদর্শন করুন
2. **Python 3.10** বা **3.11** ডাউনলোড করুন (Windows 64-bit)
3. ইনস্টলার চালান এবং **"Add Python to PATH" চেক করুন**
4. "Install Now" ক্লিক করুন

ইনস্টলেশন যাচাই করুন:
```bash
python --version
```

### ধাপ 2: অ্যাপ্লিকেশন ডাউনলোড করুন

**বিকল্প A: Git ব্যবহার করে (সুপারিশকৃত)**
```bash
git clone https://github.com/adibgafur/accessibility-automation-agent.git
cd accessibility-automation-agent
```

**বিকল্প B: ZIP ডাউনলোড করুন**
1. [GitHub Repository](https://github.com/adibgafur/accessibility-automation-agent) পরিদর্শন করুন
2. "Code" → "Download ZIP" ক্লিক করুন
3. ZIP ফাইল আপনার পছন্দের স্থানে এক্সট্র্যাক্ট করুন

### ধাপ 3: নির্ভরতা ইনস্টল করুন

প্রকল্প ডিরেক্টরিতে কমান্ড প্রম্পট খুলুন এবং চালান করুন:

```bash
pip install -r requirements.txt
```

এটি ইনস্টল করবে:
- PyQt6 (ব্যবহারকারী ইন্টারফেস)
- Whisper (ভয়েস স্বীকৃতি)
- MediaPipe (চোখের ট্র্যাকিং)
- Selenium (ব্রাউজার অটোমেশন)
- OpenCV (কম্পিউটার ভিশন)
- এবং অন্যান্য নির্ভরতা

**নোট**: প্রথম-সময়ের সেটআপ 10-15 মিনিট সময় লাগতে পারে কারণ বড় মডেল ডাউনলোড করা হয়।

### ধাপ 4: ML মডেল ডাউনলোড করুন

প্রথম চালানোতে, অ্যাপ্লিকেশন স্বয়ংক্রিয়ভাবে ডাউনলোড করবে:
- Whisper মডেল (1.4GB) - ভয়েস স্বীকৃতির জন্য
- MediaPipe মডেল (~50MB) - চোখের ট্র্যাকিংয়ের জন্য
- GUIrilla মডেল (~2GB) - GUI সনাক্তকরণের জন্য

**মোট: ~3.5GB** - নিশ্চিত করুন যে আপনার কাছে যথেষ্ট ইন্টারনেট ব্যান্ডউইথ এবং ডিস্ক স্থান রয়েছে।

### ধাপ 5: ইনস্টলেশন যাচাই করুন

নিশ্চিত করতে টেস্ট স্যুট চালান:

```bash
python -m pytest tests/ -v
```

সমস্ত টেস্ট পাস হওয়া উচিত। যদি ব্যর্থতা থাকে, [সমস্যা সমাধান](#সমস্যা-সমাধান) দেখুন।

## শুরু করা

### প্রথম সময় সেটআপ

#### 1. অ্যাপ্লিকেশন লঞ্চ করুন

**GUI মোড** (সুপারিশকৃত):
```bash
python -m src.main --language bn
```

**হেডলেস মোড** (শুধুমাত্র ভয়েস + চোখের ট্র্যাকিং):
```bash
python -m src.main --headless --language bn
```

#### 2. চোখের ট্র্যাকার ক্যালিব্রেট করুন

1. "Eye Tracking" ট্যাব ক্লিক করুন
2. "Calibration শুরু করুন" ক্লিক করুন
3. অন-স্ক্রিন নির্দেশাবলী অনুসরণ করুন
4. ক্যালিব্রেশন পয়েন্টগুলিতে তাকান যখন তারা প্রদর্শিত হয়
5. ক্যালিব্রেশন সিকোয়েন্স সম্পূর্ণ করুন

#### 3. ভয়েস কমান্ড পরীক্ষা করুন

1. "Voice Control" ট্যাব ক্লিক করুন
2. "শোনা শুরু করুন" ক্লিক করুন
3. একটি সহজ কমান্ড বলুন যেমন "Click" বা "Open Notepad"
4. সিস্টেম কমান্ড সম্পাদন করবে

#### 4. সেটিংস কনফিগার করুন

1. "Settings" ট্যাবে যান
2. আপনার পছন্দ বেছে নিন:
   - **ভাষা**: ইংরেজি বা বাংলা
   - **থিম**: হালকা, গাঢ়, বা উচ্চ বৈপরীত্য
   - **অডিও প্রতিক্রিয়া**: পাঠ্য-থেকে-কথা সক্ষম/অক্ষম করুন
   - **মাইক্রোফোন**: আপনার মাইক্রোফোন ডিভাইস নির্বাচন করুন
   - **ক্যামেরা**: চোখের ট্র্যাকিংয়ের জন্য আপনার ক্যামেরা নির্বাচন করুন

## বৈশিষ্ট্য এবং ব্যবহার

### 1. ভয়েস নিয়ন্ত্রণ

#### ভয়েস কমান্ড

| কমান্ড | ফাংশন |
|--------|--------|
| **"Click"** | বর্তমান চোখের দিকে তাকানোর অবস্থানে ক্লিক করুন |
| **"Double click"** | বর্তমান অবস্থানে ডাবল-ক্লিক করুন |
| **"Right click"** | বর্তমান অবস্থানে রাইট-ক্লিক করুন |
| **"Search [শব্দ]"** | দেওয়া শব্দের জন্য ওয়েব অনুসন্ধান করুন |
| **"Open [অ্যাপ্লিকেশন নাম]"** | একটি অ্যাপ্লিকেশন লঞ্চ করুন |
| **"Record macro"** | একটি ম্যাক্রো রেকর্ডিং শুরু করুন |
| **"Stop recording"** | বর্তমান ম্যাক্রো রেকর্ডিং বন্ধ করুন |
| **"Play [ম্যাক্রো নাম]"** | একটি সংরক্ষিত ম্যাক্রো পুনরাবৃত্তি করুন |

#### কাস্টম ভয়েস কমান্ড

কাস্টম ভয়েস কমান্ড যোগ করতে:
1. `config/default_settings.yaml` সম্পাদনা করুন
2. আপনার কমান্ড `voice_commands` বিভাগে যোগ করুন
3. অ্যাপ্লিকেশন পুনরায় চালু করুন

উদাহরণ:
```yaml
voice_commands:
  custom_email:
    - "open email"
    - "open gmail"
    action: "open_app('Gmail')"
```

### 2. চোখের ট্র্যাকিং

#### বৈশিষ্ট্য

- **মসৃণ ট্র্যাকিং**: রিয়েল-টাইম চোখের অবস্থান ট্র্যাকিং
- **পলক সনাক্তকরণ**: ক্লিকিংয়ের জন্য একক বা ডাবল পলক
- **ক্যালিব্রেশন**: নির্ভুলতার জন্য 9-পয়েন্ট ক্যালিব্রেশন
- **জিটার ফিল্টারিং**: চোখের কাঁপুনি থেকে কার্সার কম্পন কমায়

#### ব্যবহার

1. UI থেকে চোখের ট্র্যাকিং সক্ষম করুন
2. প্রয়োজন হলে ক্যালিব্রেশন সম্পূর্ণ করুন
3. স্ক্রিনের দিকে তাকান কার্সার নড়াতে
4. বর্তমান অবস্থানে ক্লিক করতে চোখ ঝাপসা করুন

### 3. ম্যাক্রো রেকর্ডিং এবং পুনরাবৃত্তি

#### ম্যাক্রো রেকর্ডিং

1. "Macro" ট্যাব ক্লিক করুন
2. ম্যাক্রো নাম প্রবেশ করুন (যেমন "email_workflow")
3. "রেকর্ডিং শুরু করুন" ক্লিক করুন
4. আপনার ক্রিয়াকলাপ সম্পাদন করুন (ক্লিক, টাইপিং, নেভিগেশন)
5. "রেকর্ডিং বন্ধ করুন" ক্লিক করুন
6. আপনার ম্যাক্রো স্বয়ংক্রিয়ভাবে সংরক্ষিত হয়

#### ম্যাক্রো পুনরাবৃত্তি করুন

1. তালিকা থেকে একটি সংরক্ষিত ম্যাক্রো নির্বাচন করুন
2. পুনরাবৃত্তির গতি সেট করুন (0.5x থেকে 2x)
3. লুপ গণনা সেট করুন (কত বার পুনরাবৃত্তি করতে হবে)
4. "ম্যাক্রো চালান" ক্লিক করুন
5. ক্রিয়াগুলি পুনরাবৃত্তি হতে দেখুন

#### 2. ভয়েস স্বীকৃতি কাজ করছে না

**সমস্যা**: ভয়েস কমান্ড স্বীকৃত নয়

**সমস্যা সমাধান**:
1. মাইক্রোফোন নিঃশব্দ নয় তা পরীক্ষা করুন
2. Windows সেটিংসে মাইক্রোফোনের অনুমতি পরীক্ষা করুন
3. মাইক্রোফোন ভলিউম পর্যাপ্ত (কমপক্ষে 50%)
4. উপলব্ধ থাকলে অন্য মাইক্রোফোন চেষ্টা করুন
5. সেটিংসে ভাষা সঠিকভাবে সেট করুন
6. আত্মবিশ্বাসের থ্রেশহোল্ড খুব বেশি নয় তা পরীক্ষা করুন

#### 3. চোখের ট্র্যাকিং সমস্যা

**সমস্যা**: চোখের কার্সার ঝাপসা বা অনুচিত

**সমাধান**:
1. পুনরায় ক্যালিব্রেট করুন: চোখের ট্র্যাকিং ট্যাবে "ক্যালিব্রেট" ক্লিক করুন
2. পর্যাপ্ত আলো নিশ্চিত করুন
3. ক্যামেরা লেন্স পরিষ্কার করুন
4. ভাল কোণের জন্য ক্যামেরার অবস্থান সামঞ্জস্য করুন
5. কনফিগে `smoothing_factor` বৃদ্ধি করুন (0.5-0.9)
